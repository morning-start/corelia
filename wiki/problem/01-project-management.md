### 应用搜索功能说明

**功能定位**：集成到文件搜索插件，提供快速打开本地应用的能力

**用户场景**：
- 输入 `vs` → 快速启动 Visual Studio Code
- 输入 `chrome` → 快速启动 Google Chrome
- 输入 `wechat` → 快速启动微信

**技术实现**：
- 索引 Windows "开始菜单" 和 "桌面" 快捷方式
- 使用 `tiny-fuzzy` 模糊匹配
- 调用系统 shell 打开应用

### WASM 补丁依赖

| 补丁名称 | 说明 | MVP 中用途 |
|----------|------|------------|
| **crypto** | 加密算法 | 二维码生成、数据加密 |
| **AI** | AI 模型集成 | 截图 OCR、图像识别 |
