## 3.4 中文分词方案

**决策**：**A 方案（前端分词）**，使用 `tiny-fuzzy`

| 问题 | 决策 | 说明 |
|------|------|------|
| 分词库 | **`tiny-fuzzy`** | 轻量，支持模糊匹配 |
| 拼音支持 | **前端实现** | MVP 够用 |
| 用户词典 | **MVP 不支持** | 后续扩展 |
| 性能要求 | **< 10ms** | 毫秒级响应 |

**技术实现**：
```
用户输入 "wendang"
    │
    ▼
tiny-fuzzy 模糊匹配
    │
    ├── "文档" ✅
    ├── "我的文档" ✅
    └── "文旦" ✅
```

**已确定** ✅：
- [x] 分词库 → **`tiny-fuzzy`**
- [x] MVP 精度 → **够用即可**

## 3.5 主题系统设计

**决策**：按建议方案

| 问题 | 决策 | 说明 |
|------|------|------|
| 主题变量 | **CSS Custom Properties** | 现代 CSS 方案 |
| 内置主题 | **3 个（深色/浅色/跟随系统）** | 满足基本需求 |
| 自定义主题 | **MVP 不支持** | 后续扩展 |
| 实时切换 | **支持** | 无需重启 |

**主题变量设计**：
```css
:root {
  --color-bg: #1a1a1a;
  --color-text: #ffffff;
  --color-accent: #0078d4;
  --radius: 8px;
  /* ... */
}
```

**已确定** ✅：
- [x] 主题变量 → **CSS Custom Properties**
- [x] 内置主题 → **3 个（深/浅/跟随）**
