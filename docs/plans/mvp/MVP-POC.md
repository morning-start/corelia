## 目标与验收标准

### 目标

验证以下核心技术的可行性：

1. **Tauri 2.x 窗口管理**：无边框窗口、透明背景、置顶显示
2. **全局快捷键**：Alt+Space 唤起窗口
3. **QuickJS 集成**：JS 引擎运行和通信
4. **WASM 基础**：wasm-pack 编译和调用
5. **前端搜索**：模糊匹配性能

### 验收标准

| 序号 | 验收标准 | 验证方法 | 通过标准 | 状态 |
|------|---------|---------|---------|------|
| 1 | Tauri 窗口可在无边框模式下运行 | 运行测试 | 窗口正常显示 | ✅ |
| 2 | 窗口支持透明背景和圆角 | 视觉检查 | 背景透明、圆角正常 | ✅ |
| 3 | 全局快捷键 Alt+Space 可唤起窗口 | 手动测试 | 窗口正常唤起 | ✅ |
| 4 | QuickJS VM 可正常实例化 | 运行测试 | 无错误日志 | ⚠️ |
| 5 | QuickJS 可调用 Tauri Command | 运行测试 | 数据正常返回 | ⚠️ |
| 6 | WASM 模块可正常加载 | 运行测试 | 函数正常调用 | ✅ |
| 7 | fuzzy 搜索响应 < 50ms | 性能测试 | 1000 条数据下 | ✅ |

### 详细任务说明

#### POC-01 Tauri 项目初始化

**任务描述**：创建基于 Tauri + SvelteKit + TypeScript 的基础项目

**执行步骤**：
1. 使用官方脚手架创建项目
2. 配置 Bun 作为包管理器
3. 验证项目可运行
4. 配置 Git 仓库

**状态**：✅ 已完成

#### POC-02 窗口配置

**任务描述**：配置 Tauri 窗口为无边框，透明、置顶模式

**执行步骤**：
1. 修改 tauri.conf.json 窗口配置
2. 实现透明背景 CSS
3. 实现圆角样式
4. 实现窗口置顶
5. 实现点击外部隐藏

**状态**：✅ 已完成

#### POC-03 全局快捷键实现

**任务描述**：实现 Alt+Space 全局快捷键唤起窗口

**执行步骤**：
1. 安装 tauri-plugin-global-shortcut
2. 注册 Alt+Space 快捷键
3. 实现快捷键处理逻辑
4. 测试失去焦点后的响应

**状态**：✅ 已完成

#### POC-04 QuickJS 集成验证

**任务描述**：验证 QuickJS 引擎可正常运行 JavaScript

**执行步骤**：
1. 引入 QuickJS 库
2. 实现 VM 实例化
3. 执行简单 JS 代码
4. 验证 JS 运行环境正常

**状态**：⚠️ 使用模拟模式（Windows MSVC 编译问题）

#### POC-05 Tauri Event 通信验证

**任务描述**：验证前端与 Rust 通过 Event 通信

**执行步骤**：
1. 实现前端 Event 监听
2. 实现 Rust Event 发送
3. 实现 Command 调用
4. 验证双向通信正常

**状态**：✅ 已完成

#### POC-06 WASM 基础环境搭建

**任务描述**：搭建 WASM 开发环境，验证 wasm-pack 可用

**执行步骤**：
1. 配置 Rust WASM 工具链
2. 创建 WASM lib 项目
3. 编写简单 Rust 函数
4. 使用 wasm-pack 编译
5. 前端加载和调用

**状态**：✅ 编译成功（使用浏览器原生 API 降级）

#### POC-07 fuzzy 搜索测试

**任务描述**：验证 fuzzy 在前端搜索中的性能

**执行步骤**：
1. 安装 fuzzy 库
2. 创建 1000 条测试数据
3. 测量搜索响应时间
4. 验证 < 50ms 目标

**状态**：✅ 已完成（实测约 8ms）

#### POC-08 原型报告编写

**任务描述**：编写技术原型验证报告

**输出内容**：
- 各技术验证结果
- 遇到的问题及解决方案
- 性能测试数据
- 后续建议

**状态**：✅ 已完成
