## 架构概览

Corelia 采用 **Tauri 2.x + Svelte 5 + Rust** 技术栈，实现轻量、安全、可扩展的桌面效率工具。

### 核心设计原则

1. **前后端分离** - UI在前端，系统能力在后端
2. **三层插件架构** - QuickJS + Webview + WASM
3. **懒加载设计** - 按需加载，保证性能
4. **充分解耦** - 为未来优化留足空间
